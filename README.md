# Chat Application - React Chat Engine

Một ứng dụng chat thời gian thực đơn giản được xây dựng bằng **React** và thư viện **[react-chat-engine](https://github.com/chatengine-io/react-chat-engine)**.  
Ứng dụng hỗ trợ đăng nhập, gửi tin nhắn text, gửi hình ảnh, hiển thị trạng thái đang gõ (typing), và read receipts.

## ✨ Tính năng chính

- Đăng nhập / Đăng ký qua username & password (lưu vào localStorage)
- Giao diện chat hiện đại, responsive
- Gửi tin nhắn text và hình ảnh
- Hiển thị tin nhắn của bản thân và người khác
- Read receipts (dấu tick đã đọc)
- Âm thanh thông báo khi có tin nhắn mới
- Custom styling đẹp mắt

## 🛠 Công nghệ sử dụng

- **Frontend**: React 17 + React Scripts (Create React App)
- **Chat Backend**: [Chat Engine](https://chatengine.io) (projectID: `1b7801d6-8a66-4be4-a442-89219d833dfc`)
- **UI Icons**: `@ant-design/icons`
- **HTTP Client**: Axios
- **Styling**: CSS modules + custom styles
